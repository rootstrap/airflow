1. Move to chart directory 
   cd chart
2. Create namespace
3. Set kubernetes namespace as default: kubectl config set-context --current --namespace=default
4. Add repo 
   helm repo add stable https://charts.helm.sh/stable/
5. Install cluster
   helm install airflow . --namespace default
6. helm upgrade airflow . --namespace default

Forwarding to access the web:  kubectl port-forward $(kubectl get pods | awk '{print $1}' | grep 'webserver') 8080:8080
