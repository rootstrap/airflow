1. Move to chart directory 
   cd chart
2. Create namespace
   kubectl create namespace airflow
3. Set kubernetes airflow namespace as default kubectl config set-context --current --namespace=airflow
4. Add repo 
   helm repo add stable https://charts.helm.sh/stable/
5. Install cluster
   helm install airflow . --namespace airflow

Forwarding to access the web:  kubectl port-forward $(kubectl get pods | awk '{print $1}' | grep 'webserver') 8080:8080
